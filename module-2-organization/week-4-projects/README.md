# Week 8: Module 2 Projects

## Tổng Quan (Overview)

Tuần cuối của Module 2 tập trung vào **hands-on projects** để consolidate knowledge về test organization, Page Object Model, fixtures, và multi-environment configurations.

## Projects

### Project 1: Refactor to Page Object Model
**Location:** `project-01-refactor-to-pom/`
**Objective:** Refactor existing tests to use Page Object Model pattern

**What You'll Learn:**
- Converting procedural tests to POM
- Creating page classes with proper encapsulation
- Using fixtures for page objects
- Organizing tests with clean architecture

### Project 2: Multi-browser Test Suite
**Location:** `project-02-multi-browser-suite/`
**Objective:** Build a comprehensive test suite that runs on multiple browsers and environments

**What You'll Learn:**
- Configuring multi-browser projects
- Environment-specific configurations
- Cross-browser compatibility testing
- Performance comparison across browsers

### Project 3: CI/CD Integration
**Location:** `project-03-ci-setup/`
**Objective:** Set up continuous integration with GitHub Actions

**What You'll Learn:**
- Creating GitHub Actions workflows
- Running tests in CI/CD pipeline
- Artifact collection (screenshots, videos, reports)
- Parallel execution in CI
